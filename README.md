# Chocolate Company Sales & Shipments Dashboard

This project analyzes sales and shipment data for a chocolate company using Power BI.

## Tools Used
- Power BI
- Excel
- DAX
- Data Visualization

## Dashboard Features
- Total Sales KPI
- Profit Analysis
- Sales by Region
- Monthly Sales Trend
- Product Performance

## Dataset
Chocolate company sales and shipments data.
